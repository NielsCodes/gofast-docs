## Configuration

This choice must be done using GoFast CLI.

- We heavly utilize SSR.
- We keep the frameworks up to date.
- We use the suggested approach, `Form Action` for SvelteKit and `Server Action` for NextJS.
- The logic is extracted to `server/services` folder.
